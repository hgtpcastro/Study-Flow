# 3. Criando o gitignore de forma simples
- [Pacote do gitignore](https://www.npmjs.com/package/gitignore)

# Comandos

# Ferramenta para gerenciar o .gitignore
```
npm install gitignore -g
```

# Lista os tipos
```
gitignore -types
```

# Para prohjetos JS
```
gitignore Node
```

# Visualiza o arquivo
```
more .gitignore
```
